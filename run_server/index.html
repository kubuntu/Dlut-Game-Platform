如何运行“地瓜皮”服务器端

首先，请确定你已经完成DGP的编译

如何编译“地瓜皮”项目源代码

运行主服务器

  • 在ubuntu linux 10.10下
    • 双击“ServerBuildRun.sh”，选择“在终端中运行”，启动主服务器。
  • 其它操作系统请自行研究。

记录端口号

  • 记录下“freeport Server : port is ”后面的数字。我的是37373。
    • （如果你的不是37373，后面的SnakeServer和GameFileServer1将会启动失败，暂时我也没有解决办法。）
  • 您将您的IP地址及freeport Server的端口提供给游戏玩家，他们就可以登录您的DGP服务器来进行游戏了。

运行SnakeServer与GameFileServer1

  • 在ubuntu linux 10.10下
    • 双击“SnakeServerBuildRun.sh”，选择“在终端中运行”，启动SnakeServer
    • 双击 “GameFileServer1BuildRun.sh”，选择“在终端中运行”，启动GameFileServer1
  • 其它操作系统请自行研究。

让其他玩家登录您的DGP服务器来进行游戏

  • 首先，您可以打开Dlut-Game-Platform/database/login文件，添加一名用户。
    • 第一列是用户ID，请不要与其它用户重复，请选择一个不超过65535的正整数。
    第二列是登录名，请不要与其它用户重复并且不能包含空格。
    第三列是密码，请不要包含空格。
    第四列是这名用户拥有的权限，1是游戏玩家，2是游戏开发者，4是子服务器运营者，8是平台管理员，16是根用户。如果一名用户同时拥有多个权限，则此处应填写这些权限的简单加和。
  • 然后，关闭并重新启动MainServer、SnakeServer与GameFileServer1。
  • 最后，将您的IP地址、freeport Server的端口、用户名、密码及权限通知给玩家，他们就可以登录您的服务器进行游戏了。

全部

以下内容能够运行在ubuntu linux 10.10下，其它操作系统请自行研究。
关闭主服务器，关闭主客户端，关闭SnakeServer与GameFileServer1。
然后双击“AllRun.sh”，选择“在终端中运行”。
