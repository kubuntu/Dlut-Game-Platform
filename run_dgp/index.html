如何运行“地瓜皮”

首先，请确定你已经完成DGP的编译

如何编译“地瓜皮”项目源代码

运行主服务器

在ubuntu linux 10.10下，直接双击“ServerBuildRun.sh”，选择“在终端中运行”即可。
其它操作系统请自行研究。
记录下“freeport Server : port is ”后面的数字。我的是37373。（如果你的不是37373，后面的SampleServer和GameFileServer1将会启动失败，暂时我也没有解决办法。）

运行主客户端

在ubuntu linux 10.10下，直接双击“ServerBuildRun.sh”，选择“在终端中运行”即可。
其它操作系统请自行研究。
在User name与Pass word中填入用户名和密码。
目前仅有一名用户。用户名为“elephant”，密码为“acm”。（请注意，全部为小写。）
点击server -> new server，name不用管，地址填入“127.0.0.1”（没有引号），端口填入主服务器中“freeport Server : port is ”后面的数字，我的是37373.
点击OK，登录成功

运行SampleServer与GameFileServer1

在ubuntu linux 10.10下，双击“SampleServerBuildRun.sh”，选择“在终端中运行”;双击“GameFileServer1BuildRun.sh”，选择“在终端中运行”
其它操作系统请自行研究。

刷新游戏列表并启动游戏

切换回主客户端，点击“refresh list”，选择“sample game”。
点击“start game”，它会提示询问是否下载游戏，选择“Yes”。
它会提示询问选择哪个下载服务器。目前我们只有一个下载服务器。保持默认值即可，选择“OK”。
游戏下载完成后便会自动启动。
多人贪食蛇游戏：ASDF控制右下角粉色的蛇，上下左右控制左上角绿色的蛇。另外两条蛇暂时无人控制（自生自灭）。

全部

以下内容能够运行在ubuntu linux 10.10下，其它操作系统请自行研究。
关闭主服务器，关闭主客户端，关闭SampleServer与GameFileServer1。
然后双击“AllBuildRun.sh”，选择“在终端中运行”。
