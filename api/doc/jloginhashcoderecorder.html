    Detailed Description

    管理登录用户ID及Login hash code。

    用户登录时，首先客户端向服务器发送用户名和密码，然后服务器对用户名和密码进行验证。验证通过后，服务端会依据某种规则产生一个Login hash code，并将用户的ID与这个Login hash code发送给客户端。 在客户端向服务器请求端口信息、用户信息等服务时，需提供用户ID和Login hash code。服务器端会首先对用户ID与Login hash code进行验证。验证通过后服务器端才会向客户端提供各种服务，否则立即断开连接。
