    Detailed Description

    登录请求。

    JRequestLogin对登录请求提供一个基本的封装。 登录的过程： 1、通过 JRequestPort 获得登录服务的端口。 2、调用 connectToHost() 函数连接服务器。 3、调用 login() 函数发送用户名、密码及登录身份。
