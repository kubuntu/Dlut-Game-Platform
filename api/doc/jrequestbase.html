    Detailed Description

    向服务器发送请求的基类。

    客户端向服务器发送各种请求，需要完成各种功能。 请求将数据与底层的套接字分开，对套接字进行了一定的封装。 JRequestBase封装了一此各种请求都需要用到的功能。
