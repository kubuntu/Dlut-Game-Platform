    Detailed Description

    管理版本号。

    一个版本号应包含四部分：<主版本号>.<次版本号>.<发布号>-<开发号> 每一部分应该是一个不大于255的整数。因此版本号可以保存在一个quint32 类型中。

    Member Function Documentation

    JVersion::JVersion ( quint32 data = 0xFFFFFFFF )

    以data 构造一个版本号。 默认将构造一个空版本号。

    See also isNull().

    quint32 JVersion::getData () const

    直接获取内部保存的quint32 值。

    bool JVersion::isNull () const

    判断当前版本号是否为空。 如果保存的数据等于0xFFFFFFFF，则为空。

    bool JVersion::operator!= ( const JVersion & b ) const

    如果当前版本号不等于b ，返回true；否则返回false。

    bool JVersion::operator< ( const JVersion & b ) const

    如果当前版本号低于b ，返回true；否则返回false。

    bool JVersion::operator== ( const JVersion & b ) const

    如果当前版本号等于b ，返回true；否则返回false。

    bool JVersion::operator> ( const JVersion & b ) const

    如果当前版本号高于b ，返回true；否则返回false。
