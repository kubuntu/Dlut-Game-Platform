    Detailed Description

    需要Login hash confirm的请求的基类。

    客户端向服务器发送的部分请求需要通过Login hash confirm。这个类是这类请求的基类，对Login hash confirm的过程提供了封装。 当已经连接成功后，需要调用 sendLoginHashCode() 发送Login hash code。 之后可以监听lhcResult() 信号或调用waitForLhc() 函数来获得Login hash confirm的结果。

    See also JLoginHashCodeRecorder and JRequestBase.

    void JLhcRequestBase::sendLoginHashCode ( JID userId, const QByteArray & code )

    发送Login hash code 。 userId 表示当前用户的ID，code 表示Login hash code。 如果套接字为空或已经通过Login hash confirm，函数将直接返回。 调用此函数后， getLhcState() 将变为 EPS_Sending 。

    bool JLhcRequestBase::waitForLhc ( int msecs = 30000 ) const

    等待msecs 毫秒或收到Login hash confirm的结果。 返回true ： Login hash confirm成功。 返回false ： Login hash confirm失败或时间超过msecs 毫秒。
