如何向DGP贡献代码

首先，请在github.com上注册账号并登录

然后，fork DGP的版本库

  • 打开DGP项目的源代码的托管地址：https://github.com/lexdene/Dlut-Game-Platform
  • 点击页面右上角的fork，则会在您自己的账户的目录下产生一个DGP的fork代码库。

第三步，在您自己的代码库中进行修改

  • 使用git工具管理您的代码库。
    • git工具的学习资料请见：git学习资料
  • 您可以对DGP的代码进行修改并上传到github.com上您自己的代码库中。

最后，请求将您的代码库合并到DGP的主代码库中。

  • 当您认为您的代码库比较成熟后，点击github.com上您的代码库页面上的pull request，向DGP主仓库发出一个“拉取请求”。
  • DGP项目的开发人员在对您的代码审查通过后，会将您的代码合并到DGP项目的主代码库中。
