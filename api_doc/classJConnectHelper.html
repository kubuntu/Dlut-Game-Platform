void JConnectHelper::connectToHost ( const SHost &  host  ) 

连接到地址为host 服务器。
若套接字为空，则发射 error() 信号、从 QDebug 输出错误提示并立即返回。
若已经连接，则从 QDebug 输出错误提示并立即返回。
调用此函数后， getConnectState() 将变为 ECS_Connecting 。
