详细描述

封装请求用户信息的底层通信

可以请求任何一名用户的用户信息。

注解:
DGP API 提供了更加易于使用的 JRequestUserInfo 类来处理登录请求。

通信协议

首先，这个通信协议只定义了基本通信协议 中data的部分。
查看完整的通信协议，请见基本通信协议 。

  • 客户端向服务器端 : <JID:userId>
    1. JID 类型，表示请求的用户ID。
  • 服务器端向客户端 ：<JID:userId> <UserInfo::SUserInfo:userinfo>
    1. JID 类型，表示用户ID。
    2. UserInfo::SUserInfo 类型，表示用户信息。格式化方式参见 operator>>(QDataStream& stream,UserInfo::SUserInfo& ui)
