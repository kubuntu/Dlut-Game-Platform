封装请求端口列表的底层通信

客户端向服务器端请求各个服务的端口。

注解:
DGP API 提供了更加易于使用的 JRequestPort 类来处理登录请求。

通信协议

首先，这个通信协议只定义了基本通信协议 中data的部分。
查看完整的通信协议，请见基本通信协议 。
