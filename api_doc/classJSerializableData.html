详细描述

可序列化的数据的抽象基类

可序列化对象需要实现三个抽象方法：

  • fromByteArray() 从二进制数据中序列化出对象数据
  • toByteArray() 将对象数据序列化为二进制数据
  • head() 对象的头信息。通过头信息索引数据。
