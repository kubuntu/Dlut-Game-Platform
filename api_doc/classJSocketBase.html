void JSocketBase::on_socket_readyRead (  )  [private, slot]

处理数据

数据开头是2字节的协议类型和4字节的数据长度，
分别是 JType 类型和 qint32 类型。
之后是数据内容。

首先交给 JSocketStrategy 对象认证，
认证通过后交给 JNetworkDataProcessorBase 对象处理。
