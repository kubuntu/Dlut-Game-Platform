如何运行“地瓜皮”客户端

首先，请确定你已经完成DGP的编译

如何编译“地瓜皮”项目源代码

运行客户端

在ubuntu linux 10.10下，直接双击“ClientBuildRun.sh”，选择“在终端中运行”即可。
其它操作系统请自行研究。

登录用户名、密码、身份、服务器地址和端口

  • 请联系平台运维者，索要自己的用户名、密码、身份以及服务器地址和端口。
    • 或者您可以自己运行DGP的服务器端，参见如何运行“地瓜皮”服务器端。
  • 在登录窗口中，填入自己的用户名（User Name）、密码（Pass word）。选择自己的身份（role）。
  • 在server中选择new server，并填入服务器地址和端口。
  • 点击OK登录DGP。

选择一个游戏，点start开始下载游戏并运行
