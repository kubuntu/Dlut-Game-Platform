介绍

DGP使用git工具托管在github.com上。

全新下载

如果您从未下载过DGP的源代码，您需要全新下载DGP源代码。
下面举例ubuntu linux 10.10下如何使用git工具全新下载DGP源代码，其它平台类似。
打开终端，输入：
git clone https://github.com/lexdene/Dlut-Game-Platform.git
或者
git clone git://github.com/lexdene/Dlut-Game-Platform.git
在大连理工大学校园内，使用第一个地址速度会快一些。如果您在网吧，使用第二个地址速度会快一些。
稍等片刻，即下载完毕。

更新下载

如果您曾经下载过DGP的源代码，但是您保存的代码略旧，您需要一份最新的DGP源代码，那么您需要更新下载
下面举例ubuntu linux 10.10下如何使用git工具更新下载DGP源代码，其它平台类似。
打开终端，输入：
git pull https://github.com/lexdene/Dlut-Game-Platform.git master
或者
git pull git://github.com/lexdene/Dlut-Game-Platform.git master
在大连理工大学校园内，使用第一个地址速度会快一些。如果您在网吧，使用第二个地址速度会快一些。
稍等片刻，即下载完毕。
