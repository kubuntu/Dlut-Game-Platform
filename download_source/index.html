如何下载“地瓜皮”项目源代码

介绍

DGP使用git工具托管在github.com上。

源代码地址为：https://github.com/lexdene/Dlut-Game-Platform
git仓库地址为：https://github.com/lexdene/Dlut-Game-Platform.git
git工具的学习资料请见：git学习资料

全新下载

如果您从未下载过DGP的源代码，您需要全新下载DGP源代码。

下面举例ubuntu linux 10.10下如何使用git工具全新下载DGP源代码，其它平台类似。
打开终端，输入：
git clone https://github.com/lexdene/Dlut-Game-Platform.git
或者
git clone git://github.com/lexdene/Dlut-Game-Platform.git
在大连理工大学校园内，使用第一个地址速度会快一些。
如果您在网通或电信的线路，使用第二个地址速度会快一些。
稍等片刻，即下载完毕。

更新下载

如果您曾经下载过DGP的源代码，但是您保存的代码略旧，您需要一份最新的DGP源代码，那么您需要更新下载。

下面举例ubuntu linux 10.10下如何使用git工具更新下载DGP源代码，其它平台类似。
打开终端，首先进入DGP目录（Dlut-Game-Platform），然后输入：
git pull https://github.com/lexdene/Dlut-Game-Platform.git
或者 
git pull git://github.com/lexdene/Dlut-Game-Platform.git
在大连理工大学校园内，使用第一个地址速度会快一些。
如果您在网通或电信的线路，使用第二个地址速度会快一些。
如果您正确地设置过git remote，
例如您最初使用git clone来下载源代码，（git clone命令会自动地帮您设置好remote参数）,
则可以使用如下命令来更新： 
git pull origin
稍等片刻，即下载完毕。
