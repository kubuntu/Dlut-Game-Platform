如何编译“地瓜皮”项目源代码

安装编译环境

DGP的核心部分使用C++语言及Qt库。请先确保您的系统中有以下工具：
  • g++
  • make
  • qmake
  • moc-qt4
  • uic-qt4
    •  安装方法（以ubuntu linux 10.10为例，其它操作系统请自行研究）
    打开终端，输入：
    $ sudo apt-get install build-essential qt-sdk
    回车即可。

    编译DGP

    1、打开终端。
    2、进入DGP目录
    3、输入：
    $ find . -path '*-build-desktop' -exec make -C {} \;
    回车即可。等待约5分钟，编译完成。
