如何编译“地瓜皮”项目源代码

安装编译环境

DGP的核心部分使用C++语言及Qt库。请先确保您的系统中有以下工具：
  • g++
  • make
  • qt4开发库(这个要求4.7.0以上版本，不过据说4.6.3已经足够了)
  • libzip

安装方法（以ubuntu linux 10.10为例，其它操作系统请自行研究）

打开终端，输入：
$ sudo apt-get install build-essential qt-sdk libzip-dev
回车即可。

编译DGP

  1. 使用Qt Creator打开源代码目录下的Dlut-Game-Platform.pro文件
  2. 点击编译
  3. 等待约10分钟
